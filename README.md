# Flask Student Management System

A web-based Student Management System built using Flask with role-based authentication for students and teachers.

## Features

* Student login to view academic results
* Teacher login to add, update, and delete student records
* Role-based access control and secure authentication
* CRUD operations for managing student data

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* Jinja2

## How to Run

1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Run the application using `python app.py`
4. Open `http://127.0.0.1:5000/` in your browser
